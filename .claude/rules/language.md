# Language Policy

This project supports multiple languages by design.

## Rules

- Course folders follow the language of the course
- /wiki is shared — mixing languages is intentional
- Do not translate files unless explicitly asked
- Do not flag mixed language as an error
- Default response language: follow the language the owner writes in
- Wiki pages: use whichever language best serves the content

## Frontmatter language field

- `en` — English only
- `es` — Spanish only
- `en-es` — bilingual or mixed
