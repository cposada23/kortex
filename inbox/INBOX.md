# Inbox

Quick capture — drop anything here. Process with `/ingest`.

## How to use

- **Text entries:** Add below the line. One idea per bullet.
- **Files:** Drop PDFs, articles, or transcripts into `inbox/drop/`
- **Course-specific:** Drop into `sources/courses/[course-name]/inbox/`
- **Project-specific:** Drop into `projects/[project-name]/inbox/`

Run `/ingest` to process everything. Items move to `processed/` after ingest.
