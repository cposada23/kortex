# Schema Changelog

Version history for the Kortex system schema.

---

## v1.0 — Initial release

- Five Zone Architecture (inbox, sources, wiki, projects, output)
- Three-layer system (Sources, Wiki, Schema)
- CLAUDE.md root schema
- Frontmatter schema with distillation levels and confidence scoring
- Commands: /bridge, /bridge-out, /ingest, /lint, /safe-change
- Workflows: ingest, query, lint
- Hooks: validate-frontmatter, validate-links, validate-index
- Templates: concept, area, project-brief, session-end, new-note
- Pre-commit link validation
- Distributed inbox system
